  # Pokedex App

## Introduction

Welcome to the Pokedex App, a project developed during Valtech's V_Camp program. This application is the result of collaborative efforts between [@andradeviniicius] and [@KenjiWorkES]. The project serves as a demonstration of our skills in web development, specifically utilizing React and raw JavaScript to interact with the Pokemon API.

## Project Overview

### Objective

The primary goal of this project was to create a functional Pokedex application that allows users to explore information about various Pokemon. The successful completion of this project played a pivotal role in our promotion at Valtech.

### Technologies Used

- **React:** The core framework used for building the user interface and managing application state.
- **JavaScript:** Raw JavaScript was employed for interacting with the Pokemon API.
- **SCSS:** Styling
- **Pokemon API:** The application fetches data from the Pokemon API to dynamically populate the Pokedex with accurate and up-to-date information.

## Features

- **Pokemon Listing:** View a comprehensive list of Pokemon, including their names, images, and types.
- **Search Functionality:** Easily search for specific Pokemon by name or type.
- **Detailed Pokemon Information:** Access detailed information about each Pokemon, such as abilities, base experience, and stats.

## Getting Started

To run the Pokedex App locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone [repository-url]
   cd Pokedex
   npm install
   npm start
   ```

   The app will be accessible at `http://localhost:3000` in your browser.

## Project Structure

The project is structured as follows:

- **src:** Contains the source code for the React application.
- **public:** Includes static assets and the HTML file.

## Collaboration

This project was a collaborative effort between [@andradeviniicius] and [@KenjiWorkES]. Both team members contributed to different aspects of the application, including design, implementation, and testing.

## Future Improvements

While the project meets the requirements for the V_Camp program, there are areas for future improvements, such as:

- **Enhanced User Interface:** Incorporating a more user-friendly and visually appealing design.
- **Additional Features:** Implementing features like Pokemon evolution chains, move lists, and user authentication.

Feel free to explore and contribute to the project!
